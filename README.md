# Java Array Element Search Program

This Java program allows the user to:
- Input the size of an array
- Enter the array elements
- Enter a target element
- Find the index of the target element using Linear Search
- 
- ## How It Works
1. User enters the array size
2. User enters elements of the array
3. User enters the element to search
4. Program searches the array and prints the index where the element is found
   
## Example Output
---->
Enter size of your array: 5
Enter 5 element of your array: 10 20 30 40 50
Enter your target element: 30
Your element is found at index: 2

## Concepts Covered
- Arrays in Java
- Linear Search
- Loops and conditional statements
- User input using Scanner
